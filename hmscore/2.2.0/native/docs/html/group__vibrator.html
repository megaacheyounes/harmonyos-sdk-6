◆ StartVibrator()

int32_t StartVibrator ( const char *  effectId)

Controls this vibrator to perform a one-shot vibration with a preset vibration effect.

Parameters
effectIdIndicates the preset vibration effect, which is described in VIBRATOR_TYPE_WATCH_CROWN_STRENGTH1 VIBRATOR_TYPE_WATCH_CROWN_STRENGTH2 VIBRATOR_TYPE_WATCH_CROWN_STRENGTH3.
Returns
Returns 0 if the vibrator vibrates as expected; returns -1 otherwise, for example, the preset vibration effect is not supported.

int32_t StopVibrator ( const char *  mode)

Controls this vibrator to stop the vibration.

Parameters
modeIndicates the mode of the vibration to stop. The values can be time or preset, respectively representing a one-shot vibration effect, a preset vibration effect.
Returns
Returns 0 if the vibrator stops as expected; returns -1 otherwise.
