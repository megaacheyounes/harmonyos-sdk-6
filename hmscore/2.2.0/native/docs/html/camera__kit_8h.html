Detailed Description

Defines the entry for using camera functions.

In general, the camera workflow is as follows:

  • [Mandatory] Call CameraKit_GetInstance to obtain a CameraKit instance.
  • [Optional] Register a callback to monitor camera usage states using CameraKit_RegisterCameraDeviceCallback,and unregister the callback by CameraKit_UnregisterCameraDeviceCallback.
  • [Mandatory] Obtain all the supported logical cameras by calling CameraKit_GetCameraIds.
  • [Optional] Obtain more information about a physical camera by calling CameraKit_GetCameraInfo. The information includes the camera facing type, a physical camera belong to the logical camera, and link type of physical camera.
  • [Optional] Obtain static abilities of a logical or physical camera by calling CameraKit_GetCameraAbility and query the supported abilities and their configuration ranges through a specific ability querying function.
  • [Mandatory] Create a logical or physical camera object by calling CameraKit_CreateCamera.
  • [Mandatory] Start running the camera by calling functions provided in camera_device.
