Creates an EventRunnerNativeImplement instance in a new thread.

You should call this function or GetEventRunnerNativeObjForThread to obtain or create an EventRunnerNativeImplement instance so that the instance will be passed as an input parameter to other EventHandler-specific functions you will call.

◆ EventRunnerAddFileDescriptorListener()

int EventRunnerAddFileDescriptorListener ( const EventRunnerNativeImplement *  nativeObj,
int  fileDescriptor,
unsigned int  events,
FileDescriptorCallbacks *  fileDescriptorCallbacks 
)

Adds a file descriptor listener for the EventRunnerNativeImplement instance.

The file descriptor listener includes callbacks that can be invoked when the file descriptor is readable, writable, or closed, or when it encounters an exception.

Parameters
nativeObjIndicates the pointer to the EventRunnerNativeImplement instance obtained by calling GetEventRunnerNativeObjForThread or CreateEventRunnerNativeObj.
fileDescriptorIndicates the file descriptor for which the listener is to add.
eventsIndicates the file descriptor events, including input, output, and error events.
fileDescriptorCallbacksIndicates the pointer to the FileDescriptorCallbacks structure containing file descriptor-related callbacks. You must implement this structure.
Returns
Returns 0 if the file descriptor listener is added; returns another value otherwise.
