Detailed Description

Provides native functions for the resource manager to operate raw file directories and their raw files.

You can use the resource manager to traverse, open, seek, read, and close raw files.

Enumerator
RAW_FILE_MODE_UNDEFINED 

Information about how to access data is not specified.

RAW_FILE_MODE_RANDOM 

File chunks are read, with forward and backward seek.

RAW_FILE_MODE_SEQUENTIAL 

Files are read sequentially, with an occasional forward seek.

RAW_FILE_MODE_BUFFER 

The caller plans to ask for a read-only buffer with all data.

◆ SeekRawFile()

off_t SeekRawFile ( const RawFile *  rawFile,
off_t  offset,
int  whence 
)

Uses the 32-bit data type to seek a data read/write position based on the specified offset within a raw file.

Parameters
rawFileIndicates the pointer to RawFile.
offsetIndicates the specified offset.
whenceIndicates the new read/write position, which can be one of the following values:
0: The new read/write position is set to offset.
1: The read/write position is set to the current position plus offset.
2: The read/write position is set to the end of file (EOF) plus offset.
Returns
Returns the new read/write position if the operation is successful; returns (off_t) -1 if an error occurs.
